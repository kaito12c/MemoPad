## ⓪こんな作品にしたい！
テーマ：コーチングを気軽にできる
目的：コーチング的質問を１日１つ受けて答える
機能：
・毎日ランダムな質問が投げられる
・応えた質問を残すことができる
・日付・応えた内容・応えたしっくり度を５段階評価する
・応えた内容を後から編集可能に。
